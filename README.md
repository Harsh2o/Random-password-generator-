 Random Password Generator (React)

A simple and responsive Random Password Generator built using React.
It allows users to generate secure passwords by customizing length and character options, and easily copy them to the clipboard.


 Features

 Generate random passwords instantly

 Option to include numbers

 Option to include special characters

 Adjustable password length (using slider)

 One-click copy to clipboard

 Clean & responsive UI



 ⚙️ How It Works

User selects the password length

User chooses whether to include:

Numbers

Special characters

App generates a random password based on selections

User can copy the password with one click
